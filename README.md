### 专辑介绍：

###### 周冬雨张一山首次为剧献声《如果我爱你》

###### 窦鹏为《春风十里》定制音乐

###### 当“温柔一山”遇上“恬静冬雨”，便催生出的电视剧《春风十里，不如你》的片头曲《如果我爱你》。这首犹如秋水和小红彼此心迹剖白的歌曲，是周冬雨、张一山第一次为电视剧献声，并由鲜少在电视剧主创名单中出现的资深音乐人窦鹏亲自操刀。淡淡伤感的旋律搭配山雨CP独特的唱腔，将秋水和小红之间相互倾心却不敢明示，只好小心翼翼地试探、倾诉的感情，诠释得动人心弦。

###### 电视剧《春风十里，不如你》是同校师兄妹张一山和周冬雨首次组成CP，由资深音乐人窦鹏亲自操刀打造的片头曲《如果我爱你》，亦是二人首次为电视剧献声。令人惊喜万分的是，在唱歌领域略显青涩的张一山和周冬雨，都难得地突破自我在歌声里展现出另一种风格。以“北京小炮儿”的形象给人留下深刻印象的张一山，在《如果我爱你》中用充满力量又不失温柔的声线，搭配周冬雨清亮、恬静的唱腔，在“你如风似雨”“你如春般气息”的你来我往中，将秋水和小红对彼此的深情融于旋律之中。

###### 由于张一山和周冬雨并非专业歌手，而情歌想要唱出动人心弦的感觉也绝非易事。因此，他们在《如果我爱你》的录制过程中，能够在短期内颠覆往日风格，并且声线达到如此之高的匹配程度，可见山雨CP之间的默契和化学反应不只限于对手戏中。让人对《春风十里，不如你》更添期待！
