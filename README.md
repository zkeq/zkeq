###### 正因为如此，电影《喜欢你》特别邀请陈绮贞演唱影片主题曲《我喜欢上你时的内心活动》。陈绮贞在华语音乐界的崛起和成功本身也反映出普通人对生活、对爱情最纯粹的追求和渴望。

###### 而邀得著名作家、导演韩寒为电影《喜欢你》主题曲填词，则是陈可辛团队又一次诚意碰撞。只要你看过韩寒的作品，他的文字没有华丽的修饰，没有无病呻吟的缥缈，却总能用最朴素的文字，娓娓道出你我青春的那些美丽与哀愁。这恰恰也是陈可辛团队对电影《喜欢你》的创作初衷——摒弃华丽的修饰，还原爱情本来的面目，用最踏实的制作、过硬的表演，为观众诚意制作今年最走心、最甜蜜的爱情电影。

###### 其实，陈绮贞和韩寒的跨界合作也是网友期待已久的大事件。韩寒不仅在网络上多次提到陈绮贞，透露一直喜欢听她的歌，还把陈绮贞写进了自己的作品《青春》里，称赞陈绮贞的音乐是“未过度商业化的比较舒服的相对高档的大众文化”。

###### 此番电影《喜欢你》成功撮合韩寒、陈绮贞隔空合作，三方强强联手，足见影片“打造非一般爱情电影”的灼热诚意及过硬品质。
