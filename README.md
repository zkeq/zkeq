### 专辑介绍：

###### 2020陈奕迅第二首派台作《是但求其爱》

###### 有人苦心经营，觅配衬，最后痛恨入骨；

###### 有人是但求其，愈伤痕，最后如鼓琴瑟。

###### 现代人谈恋爱，宁缺勿滥，抑或宁滥勿缺？是是但但，求求其其，爱得不深，分手就不太伤心？还是敢于踏出一步，在跌跌撞撞之间冒险去爱？陈奕迅(Eason)回归广东情歌之路，淡然呼出那份爱情的无形重量。

###### 爱情但求幸福快乐，世事偏没完美，明知爱情路上，有开心亦有伤心，终身承诺可能变成终生被困，面对爱情，难免怯弱，但我们始终需要爱，需要勇敢去爱。

###### 浪漫过也享受过，快乐断不可能无时无刻，激情花火后回复平淡，两颗心可能各走各路，有人苦心经营，却因了解而分开；有人是旦求其，反而感情升华。

###### 同样是爱，千百万种变化，最后结局是笑还是泪，从来没有一种方程式，一个人或是一世人，我们终生都在寻觅爱。是，但求其爱，不求伤害。

###### 「恨不得理念相同，抵不住情景交融。

###### 邂逅总于混乱时，分离却在安稳间。

###### 有人苦心经营，觅配衬，最后痛恨入骨；

###### 有人是但求其，愈伤痕，最后如鼓琴瑟。

###### 什么是爱？

###### 是否两心越加记载，五蕴越惹尘埃？

###### 是，但求其爱，不求伤害。

###### 终身的爱？

###### 众生的爱。」

###### 今年Eason接力推出第二首广东作品《是但求其爱》，由林家谦作曲，小克填词，陈奕迅和林家谦共同监制，谱出爱情路上的怯弱无奈。歌曲慢板旋律，引子以纯钢琴音乐垫底，Eason如独白般唱出「若爱是但求开心 我问 要不要求其伤心」，配合紧接的弦乐伴奏，就似凄美的爱情，编曲简单却满载铺排，情感演绎逐层推进下，Eason以最坦率真诚的声音，唱出爱情的苦思考量。

###### 《是但求其爱》，绝不是但求其唱。11月20日，全新单曲和MV全球同步上线，各地乐迷引颈以待！
